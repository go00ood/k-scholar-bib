복잡한 숫자와 의미 없는 기호로 가득한 논문 파일명 때문에 스트레스받으셨나요? 이제 클릭 한 번으로 논문 관리의 신세계를 경험하세요! 본 확장 프로그램은 국내 주요 학술 사이트에서 논문을 다운로드할 때, 논문의 메타데이터를 자동으로 추출하여 완벽한 서지 형식으로 파일명을 바꿔줍니다.

## ✨ 주요 기능
표준 서지 형식 자동 적용: 모든 논문이 아래와 같은 깔끔한 규칙으로 저장됩니다.

    저자명(발행연도), 제목, 학술지명 권(호), 학회명, 쪽수

    (예시) 김민국(2017), ‘에서’ 주어의 통사와 의미, 국어학 81, 국어학회, 145–189.pdf

스마트한 저장 경로: 파일이 섞이지 않도록 다운로드(Downloads) / 논문 폴더에 차곡차곡 저장됩니다. (※ 브라우저 기본 다운로드 설정에 따름)


## 🌐 지원 사이트 (국내 6대 학술 DB)
대한민국 연구자라면 빠질 수 없는 주요 학술 사이트를 모두 지원합니다.

RISS 
KCI 
KISS
DBpia
교보문고 Scholar 
e-Article 

## 🚀 사용 방법
  1. 지원되는 학술 사이트의 논문 상세 페이지에 접속합니다.
  2. 평소처럼 PDF 다운로드 버튼을 누릅니다.
  3. 별도의 입력 없이도 자동으로 변환된 파일명을 확인하고 저장하세요!

## 🛠️ 참고 사항
  논문 상세 페이지가 완전히 로딩된 후 다운로드해야 가장 정확한 정보를 가져올 수 있습니다.  
  정보를 찾을 수 없는 일반 사이트에서는 기존 파일명을 그대로 유지하므로 안전하게 사용할 수 있습니다.  
  작동을 하지 않는 경우 새로고침을 누르면 됩니다.
